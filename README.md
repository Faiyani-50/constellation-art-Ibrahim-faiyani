# Constellation Art

## Description

Ce projet a été réalisé en Python avec la bibliothèque Turtle dans le cadre de l'évaluation de programmation Python.

Le programme dessine une constellation représentant un animal céleste. Les différentes parties du corps sont constituées de véritables étoiles jaunes reliées par des lignes bleues lumineuses sur un fond bleu nuit. L'animal possède également une tête ronde, des oreilles pointues et un œil brillant afin d'améliorer le rendu visuel.

## Fonctionnalités

* Création d'une fenêtre graphique avec Turtle.
* Dessin automatique d'une constellation à partir de coordonnées.
* Utilisation d'étoiles à 5 branches comme points de la constellation.
* Connexion des étoiles par des lignes bleues.
* Ajout d'une tête ronde et d'oreilles pour représenter un animal.
* Ajout d'un œil blanc pour finaliser le dessin.

## Technologies utilisées

* Python 3
* Bibliothèque Turtle

## Exécution

1. Installer Python 3 sur votre ordinateur.
2. Télécharger le fichier du projet.
3. Ouvrir un terminal dans le dossier du projet.
4. Exécuter la commande :

python projet.py
